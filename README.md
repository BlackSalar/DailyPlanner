# Ежедневник (Простой консольный планировщик задач)

Это простое консольное приложение на C#, реализующее базовый ежедневник (To-Do List).  
Приложение позволяет:
- Добавлять новые задачи
- Просматривать список задач
- Отмечать задачи как выполненные
- Удалять задачи
- Сохранять задачи в файл (чтобы данные сохранялись между запусками)

Данные хранятся в файле `tasks.json` в той же папке, где запущено приложение.

## Требования
- .NET 6.0 или выше (рекомендуется последняя версия .NET SDK)

## Как запустить

### Вариант 1: Через Visual Studio
1. Откройте Visual Studio.
2. Создайте новый проект → **Console App** (.NET Core или .NET).
3. Замените содержимое файла `Program.cs` на предоставленный код.
4. Запустите проект (F5).
